# Telemetry Catalog

This catalog documents the performance, cost, and usage metrics tracked across the repository analysis pipeline.

| Metric Name | Source | Description | Expected Range |
|---|---|---|---|
| `git_clone_duration_ms` | `GitHubAnalysisOrchestrator` | Latency to shallow clone the target repository branch. | 2,000ms ã 30,000ms |

| `code_sample_duration_ms` | `GitHubAnalysisOrchestrator` | Latency of the code sampler file walks. | 100ms ã 2,000ms |

## 2. LLM Cost & Token Usage Metrics

Tracked per LLM call and aggregated under the request `correlation_id` in `AiCostTracker` to audit platform costs.

| Metric Name | Description | Calculation Formula | USD Rate (Claude 3.5 Sonnet) |
|---|---|---|---|
| `input_tokens` | Total tokens passed to Claude (base + cache read + cache write). | Provided by Anthropic API response. | N/A |
| `output_tokens` | Tokens generated by Claude. | Provided by Anthropic API response. | $15.00 / 1M tokens |
| `cache_creation_input_tokens` | Tokens written to ephemeral cache (cache write). | Provided by Anthropic API response. | $3.75 / 1M tokens |
| `cache_read_input_tokens` | Tokens read from ephemeral cache (cache hit). | Provided by Anthropic API response. | $0.30 / 1M tokens |
| `base_input_tokens` | Base input tokens (not cached or read). | `input_tokens - cache_read - cache_write` | $3.00 / 1M tokens |
| `estimated_cost_usd` | Total cost of the analysis execution. | `(base_input * 3e-6) + (cache_write * 3.75e-6) + (cache_read * 0.3e-6) + (output * 15e-6)` | Summed calculation |

## Telemetry Emission Guidelines

Metrics are emitted in structured logs (`claude_service` logger and `github_analysis_orchestrator` logger) using the following JSON fields:
```json
{
  "duration_ms": 12450,
  "input_tokens": 15400,
  "output_tokens": 1200,
  "cache_creation_input_tokens": 14000,
  "cache_read_input_tokens": 0,
  "estimated_cost_usd": 0.063250
}
```
