## 8. Bài học về cách viết prompt

### 8.1. Khi viết prompt, em/nhóm cần cung cấp thông tin gì để AI trả lời tốt hơn?

```text
Clearly describe the project requirements and expected outcome.
Provide the technology stack (e.g., React, Node.js, Express, MongoDB, VPS).
Include relevant code snippets, error messages, and screenshots when asking for help.
Specify any constraints, such as deployment environment or coding standards.
```

### 8.2. Em/nhóm đã học được gì về cách đặt câu hỏi cho AI?

```text
Specific and detailed prompts produce more accurate and useful responses.
Providing context about the project helps AI generate better solutions.
Breaking complex problems into smaller questions makes troubleshooting easier.
Sharing errors and expected behavior allows AI to identify issues more effectively.
```

### 8.3. Lần sau em/nhóm sẽ cải thiện prompt như thế nào?

```text
Write clearer and more structured prompts.
Include sufficient project context before asking questions.
Provide relevant code and error logs when troubleshooting.
Verify AI suggestions before applying them and document the successful solutions for future reference.
```
